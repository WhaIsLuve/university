## Метод прямого выбора
### Алгоритм
Среди всех элементов ищется минимальный и меняется местами с первым элементом. Минимальный ищется среди оставшихся и меняется местами со вторым и т.д.
```c++
void selection(int *m, int n) {
	for(int i=0;i < n; i++) {
		int min = m[i], index = i;
		for(int j=j+1;j<n;j++) {
			if(m[j]<min) {
				min=m[j]; 
				index=j;
			}
		}
		
	}
}
```
